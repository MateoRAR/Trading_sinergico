# Instalación y Ejecución

### Paso 1: Instalar
```bash
# Doble click en:
install.bat

# O manualmente:
cd backend
npm install
cd ../frontend
npm install
```

### Paso 2: Ejecutar Backend
```bash
# Doble click en:
start-backend.bat

# Deberías ver:
# BackTrade API server running on port 5000
# Using simulated data (no API keys required)
```

### Paso 3: Ejecutar Frontend
```bash
# Doble click en:
start-frontend.bat

# Abre en tu navegador:
http://localhost:3000
```
